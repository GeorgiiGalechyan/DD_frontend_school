Web-технологии

Логично предположить, что web-технологии, это технологии, которые используются в web. Согласно документации MDN Web Docs к web-технологиям можно отнести:

  • базовые технологии веб разработки (HTML, CSS, HTTP);
  • технологии написания сценариев (JavaScript, Веб API, Веб компоненты);
  • графика (Canvas, SVG, WebGL);
  • аудио, видео и мультимедиа;
  • прочие.

Сегодня мы кратенько рассмотрим классическую тройку веб технологий - HTML, CSS, JavaScript.
